# Landing
## About
This repo contains code for a landing page lead generator for a tutorial series.

We use Tailwind css, Next.js, Prisma, and NeonDB.

## Development
Run local dev:
```bash
npm run dev
```

Run migrations:
```bash
npx prisma migrate dev
```

Run tests:
```bash
echo "Be the change you wish to see in the world"
```
